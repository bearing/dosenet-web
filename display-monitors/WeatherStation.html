Adding Weather Data

Partner locations can set up weather stations to collect data including temperature, pressure, humidity, wind speed, hourly precipitation, and solar radiation, which can be correlated with the DoseNet radiation sensor data to learn how environmental factors can affect background radiation rates.

Our first example weather station, installed at LBL to correlate with the LBL DoseNet device.

Data from each weather station is streamed directly to Weather Underground and available for analysis.
